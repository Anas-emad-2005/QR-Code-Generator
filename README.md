# QR Code Generator

## فكرة المشروع

أداة ويب Front End فقط لتوليد QR Code من رابط عادي أو بطاقة شخصية رقمية. تعمل داخل المتصفح بدون Backend وبدون قاعدة بيانات وبدون LocalStorage.

## الجديد في هذه النسخة

تمت إضافة نوع جديد باسم بطاقة شخصية رقمية. المستخدم يدخل الاسم ورقم الهاتف وواتساب وروابط مواقع التواصل، ثم يتم توليد QR يفتح صفحة `profile.html` تعرض البيانات بشكل أنيق.

## طريقة إخفاء البيانات من الرابط

البيانات لا تظهر كنص واضح في الرابط. يتم تحويل بيانات البطاقة إلى JSON ثم ترميزها بطريقة XOR مع Base64 URL داخل جزء الرابط بعد علامة `#` بهذا الشكل:

```text
profile.html#p=encoded-data
```

استخدام علامة `#` يجعل البيانات لا ترسل إلى السيرفر عند فتح الصفحة. هذا مناسب لمشروع Front End فقط.

مهم: هذا ليس تشفيراً أمنياً كاملاً. بدون Backend لا توجد حماية حقيقية للبيانات. الهدف هنا إخفاء البيانات عن النظر المباشر داخل الرابط، وليس منع فكها نهائياً.

## المميزات

- توليد QR لرابط عادي.
- توليد QR لبطاقة شخصية رقمية.
- عرض الاسم والوصف والنبذة ورقم الهاتف وواتساب وروابط التواصل.
- إخفاء بيانات البطاقة من الرابط بشكل غير مباشر.
- فتح صفحة مختلفة حسب بيانات QR.
- دعم RTL واللغة العربية.
- تصميم Dashboard Card متجاوب.
- تغيير لون QR.
- تغيير لون الخلفية.
- تنبيه عند ضعف التباين.
- تحميل QR بصيغ PNG وJPG وSVG وPDF وDOCX.
- نسخ الرابط.
- Toast Notifications.
- زر Reset.

## المكتبات المستخدمة

- QRCodeStyling
- jsPDF
- docx.js
- Google Fonts Cairo

## طريقة التشغيل

1. افتح ملف `index.html` في المتصفح.
2. اختر نوع QR:
   - رابط عادي
   - بطاقة شخصية رقمية
3. أدخل البيانات.
4. اضغط إنشاء QR Code.
5. امسح QR بالهاتف أو اضغط فتح البطاقة عند اختيار بطاقة شخصية.

## ملاحظات مهمة

- عند تغيير بيانات البطاقة يجب إنشاء QR جديد.
- بدون Backend لا يمكن تعديل البيانات بعد طباعة QR.
- بدون Backend لا توجد إحصائيات فتح أو لوحة إدارة.
- لا تضع بيانات حساسة لأن الإخفاء هنا ليس حماية أمنية كاملة.
- حافظ على تباين قوي بين لون QR والخلفية.

5. اختر Root.
6. اضغط Save.
