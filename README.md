# QR Code Generator

## فكرة المشروع

QR Code Generator أداة ويب Front End فقط لتوليد QR Code من رابط يدخله المستخدم. تعمل بالكامل داخل المتصفح بدون Backend وبدون قاعدة بيانات وبدون LocalStorage.

## المميزات

- توليد QR Code من رابط.
- التحقق من صحة الرابط قبل الإنشاء.
- دعم اللغة العربية واتجاه RTL.
- تصميم حديث بنمط Dashboard Card.
- واجهة متجاوبة مع الموبايل والتابلت والكمبيوتر.
- تغيير لون QR من ألوان جاهزة.
- اختيار لون مخصص عبر Color Picker.
- تغيير لون الخلفية، والافتراضي أبيض.
- تنبيه عند ضعف التباين بين QR والخلفية.
- اختيار حجم QR:
  - 200x200
  - 300x300
  - 500x500
  - 800x800
- تحميل QR بعدة صيغ:
  - PNG
  - JPG
  - SVG
  - PDF
  - Word DOCX
- نسخ الرابط المدخل.
- Toast Notifications للنجاح والخطأ.
- زر Reset لإرجاع القيم الافتراضية.

## المكتبات المستخدمة

- QRCodeStyling
- jsPDF
- docx.js
- Google Fonts Cairo

## طريقة التشغيل

1. افتح ملف `index.html` مباشرة في المتصفح.
2. أدخل رابط صحيح يبدأ بـ `http://` أو `https://`.
3. اضغط زر إنشاء QR Code.
4. اختر صيغة التحميل.
5. اضغط Download.

## ملاحظات مهمة حول التباين

يجب أن يكون لون QR Code غامقاً وواضحاً مقارنة بلون الخلفية. عند اختيار لون QR قريب من لون الخلفية، قد تفشل كاميرا الهاتف في قراءة الكود. الأفضل استخدام لون QR غامق مع خلفية بيضاء أو فاتحة.

## طريقة نشر المشروع على GitHub Pages

1. أنشئ Repository جديد في GitHub.
2. ارفع الملفات الأربعة:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. افتح Settings.
4. ادخل إلى Pages.
5. اختر Branch باسم `main`.

7. اضغط Save.
